# 🚀 Synthetic Support — Landing Page

Uma landing page moderna e responsiva para assistência técnica especializada em eletrônicos.
Foco em design profissional, experiência do usuário e performance.

---

## 📸 Preview

---

## 🧠 Sobre o projeto

O **Synthetic Support** é um site institucional desenvolvido para apresentar serviços de assistência técnica premium, com foco em:

* Credibilidade visual
* Conversão de clientes (CTA estratégico)
* Experiência fluida em dispositivos móveis
* Interface moderna inspirada em sistemas tecnológicos

---

## 🛠️ Tecnologias utilizadas

* **HTML5** → Estrutura semântica
* **CSS3** → Estilização, responsividade e animações
* **JavaScript** → Interatividade (menu, formulário)
* **Google Fonts** → Tipografia moderna (Inter + Space Grotesk)

---

## 🎯 Funcionalidades

✔ Menu responsivo com botão hambúrguer
✔ Scroll suave entre seções
✔ Seção de serviços com cards interativos
✔ Formulário de orçamento funcional
✔ FAQ com `<details>` (acessível e nativo)
✔ Layout totalmente responsivo
✔ Estrutura semântica (SEO-friendly)

---

## 📱 Responsividade

O projeto foi desenvolvido com abordagem **mobile-first**, garantindo adaptação para:

* 📱 Smartphones
* 💻 Tablets
* 🖥️ Desktops

---

## 🎨 Destaques de UI/UX

* Uso de **glassmorphism** (blur + transparência)
* Hierarquia visual bem definida
* Tipografia moderna e legível
* Microinterações com hover e transições suaves
* Cores voltadas para tecnologia (dark theme)
